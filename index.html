Portfolio

Academy Portal

HTML SASS JS PHP

Using the slim framework and the MVC design pattern, our team was tasked with updating the functionality of an existing codebase. New features included the addition of an add new trainer form, display trainers page, delete trainer button and functionality to link multiple trainers to courses. This required updating the database structure, validation in javascript, php sanitisation, unit testing and continuous integration between two teams working on the codebase.

Quotlefish

HTML CSS JS

Quotlefish is a movie themed quiz game built using Javascript. This project was created as part of an agile team tasked with accessing and displaying API data for a number of movies. The quiz is a SPA that involved retrieving JSON data using fetch, manipulating the DOM and incorporated modals, intervals and event listeners to create the game functionality. Version control was managed using Git.

Seamail Email Client

HTML SASS Bootstrap REACT

An agile team project built using REACT to create an email client that received a pre-built RESTful API. Emails were displayed within the application inbox via a fetch request, dynamic rendering was utilised to display individual emails once selected. The project was styled using bootstrap and sass to meet the provided wireframes and design plans.

Dunkin' Bisquids

HTML CSS Bootstrap PHP MySQL

Dunkin' Bisquids is an object-orientated PHP project, styled with Bootstrap, that fetches and displays information about a variety of the best biscuits from a MySQL database. Created while at the iO Academy, you can select a biscuit to view more details, vote on your favorite biscuits using the dunk and flunk buttons and then see which biscuits have topped the charts as the most dunked or most flunked.

Guidebook Collection

HTML CSS PHP MySQL

An individual project to build a collection app which displayed data from a database. The app displays my collection of rock climbing guidebooks. The app is built to both add new books and delete pre-existing entries.
