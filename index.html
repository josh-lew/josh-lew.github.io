Quotlefish

HTML
CSS
JavaScript

Quotlefish is a movie themed quiz game built using Javascript. This project was created as part of an agile team tasked with accessing and displaying API data for a number of movies. Each round entails a movie quote being selected alongside three possible answers. Users can get one hint per question and must complete the game within the allocated 30second time limit.

Dunkin' Bisquids

HTML
CSS
PHP
MySQL

Dunkin' Bisquids is an object-orientated PHP project, styled with Bootstrap, that fetches and displays information about a variety of the best biscuits from a MySQL database. Created while at the IO Academy, you can select a biscuit to view more details, vote on your favorite biscuits using the dunk and flunk buttons and then see which biscuits have topped the charts as the most dunked or most flunked.
